# 📚 인하공전 과제 알리미 사용 설명서

> 처음 쓰는 사람도 5분이면 OK! 천천히 따라오세요 😊

---

## 🖥️ 시작하기 전에

이 프로그램은 **인하공업전문대학 사이버캠퍼스(cyber.inhatc.ac.kr)** 에 자동으로 접속해서  
내 과제 목록을 한눈에 보여주는 프로그램이에요.

매번 사이버캠퍼스에 로그인해서 과목마다 들어가 과제 확인할 필요 없이,  
프로그램 켜면 **전 과목 과제 현황이 한 번에** 딱 뜹니다.

---

## 1️⃣ 프로그램 실행

`InhaTCTracker.exe` 파일을 **더블클릭**해서 실행하세요.

> 💡 **Windows 보안 경고가 뜨면?**  
> "Windows의 PC 보호" 창이 뜰 수 있어요.  
> **"추가 정보" 클릭 → "실행" 클릭** 하면 됩니다.  
> 개인이 만든 프로그램이라 인증서가 없어서 그런 거니까 걱정 안 해도 돼요.

---

## 2️⃣ 로그인

프로그램을 켜면 로그인 화면이 나와요.

| 입력 항목 | 뭘 넣으면 되냐면 |
|---|---|
| **학번 (ID)** | 사이버캠퍼스 로그인할 때 쓰는 학번 |
| **비밀번호** | 사이버캠퍼스 비밀번호 |

입력 후 **🔑 로그인** 버튼을 누르세요.

### 자동 로그인 설정
로그인 버튼 왼쪽에 **"자동 로그인"** 체크박스가 있어요.  
체크하고 로그인하면 **다음에 프로그램 켤 때 자동으로 로그인** 됩니다.  
매번 학번/비밀번호 입력하기 귀찮으면 켜두세요.

> 🔒 비밀번호는 Windows 보안 저장소(keyring)에 암호화되어 저장돼요. 텍스트 파일에 그냥 저장되는 게 아니니 안심하세요.

---

## 3️⃣ 메인 화면 구성

로그인하면 과제 목록이 쫙 뜹니다. 화면 구성은 이래요.

```
┌─────────────────────────────────────────┐
│  🎓 인하공전 과제 알리미       [로그아웃] │  ← 헤더
├─────────────────────────────────────────┤
│  📋 할 일 목록  │ 📚 수강 과목  [새로고침]│  ← 탭 바
├─────────────────────────────────────────┤
│  과제 카드들...                          │  ← 과제 목록
│                                         │
├─────────────────────────────────────────┤
│  ⚠ 마감 임박 기준: [3] 일 이내  [적용]  │  ← 설정 바
├─────────────────────────────────────────┤
│  상태 메시지              cyber.inhatc  │  ← 상태 바
└─────────────────────────────────────────┘
```

---

## 4️⃣ 과제 목록 보기 (할 일 탭)

**📋 할 일 목록** 탭이 기본으로 열려요.  
아직 제출 안 한 과제들이 여기에 쫙 나열됩니다.

### 과제 카드 색상 의미

| 색상 | 의미 |
|---|---|
| 🟥 **빨간 배경** | 마감 임박! 빨리 제출해야 해요 |
| 일반 카드 | 아직 여유 있는 과제 |

### 오른쪽 배지 색상

| 배지 | 의미 |
|---|---|
| 🔴 **미제출** | 아직 제출 안 한 과제 |
| 🟡 **진행중** | 제출 기간 중인 과제 |
| 🟢 **제출완료** | 이미 제출한 과제 |
| ⚫ **마감** | 기간이 지난 과제 |
| 🚨 **임박** | 마감이 얼마 안 남은 과제 |

### 카드 클릭하면?

카드를 클릭하면 **과제 설명이 펼쳐져요**.  
다시 클릭하면 접힙니다.

### 📤 제출하기 버튼

카드 오른쪽에 **"📤 제출하기"** 버튼이 있으면 클릭하세요.  
사이버캠퍼스 과제 제출 페이지가 **브라우저에서 바로 열립니다**.

---

## 5️⃣ 수강 과목 보기

**📚 수강 과목** 탭을 누르면 내가 듣는 과목 목록이 나와요.  
과목 이름 옆에 **"미제출 N개"** 라고 표시되면 해당 과목에 아직 안 낸 과제가 있다는 뜻이에요.

과목 카드를 클릭하면 해당 과목의 **사이버캠퍼스 강의실**이 브라우저로 열립니다.

---

## 6️⃣ 마감 임박 기준 바꾸기

화면 하단에 이런 바가 있어요:

```
⚠ 마감 임박 기준: [3] 일 이내  [적용]
```

기본값은 **3일** 이에요.  
숫자를 바꾸고 **적용** 버튼을 누르면, 그 날수 이내로 마감인 과제가 빨간색으로 표시됩니다.

예를 들어 `7`로 바꾸면 **7일 안에 마감인 과제**가 전부 빨간 경고로 표시돼요.

---

## 7️⃣ 새로고침

**🔄 새로고침** 버튼을 누르면 사이버캠퍼스에서 최신 과제 정보를 다시 불러와요.  
과제를 제출하고 나서 목록이 업데이트됐는지 확인할 때 쓰세요.

> ⏳ 과목 수에 따라 10~30초 정도 걸릴 수 있어요. 기다리면 자동으로 완료됩니다.

---

## 8️⃣ 로그아웃

헤더 오른쪽 **🚪 로그아웃** 버튼을 누르면 로그아웃 됩니다.  
자동 로그인을 꺼두고 싶을 때 로그아웃 후 체크 해제하고 다시 로그인하면 돼요.

---

## ❓ 자주 묻는 질문

**Q. 프로그램 켰는데 아무것도 안 뜨고 바로 꺼져요**  
→ `InhaTCTracker.exe` 를 우클릭 → **"관리자 권한으로 실행"** 해보세요.

**Q. 로그인이 안 돼요**  
→ 사이버캠퍼스(cyber.inhatc.ac.kr)에서 직접 로그인이 되는지 먼저 확인해보세요.  
→ 인터넷 연결도 확인해 주세요.

**Q. 과제가 실제랑 다르게 나와요**  
→ 새로고침 버튼을 눌러보세요. 그래도 이상하면 사이버캠퍼스에서 직접 확인하세요.

**Q. 설정 파일은 어디에 저장돼요?**  
→ `C:\Users\내아이디\.inhatc_tracker\settings.json` 에 저장됩니다.

---

## ⚠️ 주의사항

- 이 프로그램은 **공식 앱이 아닌 개인 제작 프로그램**이에요.
- 사이버캠퍼스 서버 구조가 바뀌면 작동이 안 될 수 있어요.
- 과제 제출은 반드시 **사이버캠퍼스에서 직접** 완료 여부를 확인하세요.
- 마감 임박 알림을 **100% 신뢰하지 말고** 중요한 과제는 직접 확인하는 습관을 들이세요.
